Proyecto: CalculadoraPyto  
Autor: Stuard Zúniga  
Lenguaje: Python  
Entorno: Pyto IDE (iOS)  
Descripción: Proyecto que implementa operaciones básicas de cálculo —suma, resta y multiplicación— y aplica una prueba unitaria mediante el módulo unittest, verificando que cada función retorne el resultado correcto.  
Objetivo: Demostrar la aplicación práctica de pruebas unitarias automatizadas en un entorno móvil.  
Repositorio público: https://github.com/stuardzuniga17-tech/CalculadoraPyto
